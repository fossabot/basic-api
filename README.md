## Set Up

At first you have to add the `src/config/tokens.json` file. It has to be structures like so:

```json
{
  "MONGODB": "YOUR_MONGO_ACCESS_TOKEN_HERE",
  "TOKE_SECRET": "RANDOM_SECRET_FOR_JWT_TOKENS"
}
```

Now you can start it. See "Scripts".
